# Project Components

## Frontend Technologies
* **React.js** - *UI component library*
* **Redux** - State management solution

  * Action creators
  * Reducers

* **Express** - *Web framework*
* **MongoDB** - Database system

  * Indexing strategy
* **JWT** - *Authentication mechanism*
